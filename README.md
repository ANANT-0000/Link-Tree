# Link Tree - Manish Kushwaha

A personal link tree website designed to showcase professional profiles and provide easy access to contact information.

## Overview

This project is a responsive, single-page website that serves as a central hub for Manish Kushwaha's online presence. It includes direct links to social media, portfolio, and professional networks.

## Features

-   **Profile Information:** Displays name, title, and profile picture.
-   **Quick Links:** Direct access to:
    -   LinkedIn
    -   GitHub
    -   Portfolio
    -   Email
    -   Company Website
-   **Interactive Elements:**
    -   **QR Code Generator:** Built-in modal to generate a QR code for the page URL using the QRious library.
    -   **Share Functionality:**  Uses the Web Share API (on supported devices) or a fallback alert for sharing the link.
-   **Responsive Design:** optimized for both mobile and desktop views using Bootstrap 5.

## Technologies Used

-   **HTML5**
-   **CSS3** (with Bootstrap 5.3 & Bootstrap Icons)
-   **JavaScript**
-   **QRious** (for QR code generation)
-   **Google Fonts** (Poppins)

## How to Run

Simply open `index.html` in any modern web browser to view the page.
